DJ를 위한 정밀 트랙 분석기.

Mixlyzer는 트랙을 스캔하고 템포·키 변화를 감지하며, 매끄러운 점프를 위한 CUE 페어를 제안하고, 빠르고 정밀한 편집을 돕는 인터랙티브 에디터를 제공합니다.

복잡한 트랙을 위한 기능

장르를 막론하고 템포와 키 변화가 있는 트랙에서 안정적인 그리드가 필요할 때, 애니메이션 OST·동인음악·리듬게임·보컬로이드·VTuber 곡까지 염두에 두고 설계되었습니다.

익스포트

다른 소프트웨어와 연동

템포 세그먼트와 JumpCUE를 XML 익스포트와 JumpCUE 렌더링으로 라이브러리에 바로 내보냅니다.

일반적인 워크플로우

필요한 모든 과정은 드래그 앤 드롭으로 시작합니다.

1) 트랙 불러오기 파일을 끌어다 놓거나 라이브러리 패널에서 선택합니다.
2) 자동 분석 템포 세그먼트, 키, JumpCUE 페어를 자동으로 감지합니다.
3) 그리드 미세 조정 BPM 조정, 세그먼트 이동, 특정 구간 재분석을 흐름을 깨지 않고 수행합니다.
4) 익스포트 편집을 저장하고 XML이나 렌더링된 오디오 파일로 내보냅니다.

원하는 방식으로 Mixlyzer 실행

개발 환경에서는 uv로, 바로 사용하려면 릴리스 빌드로 시작하세요.

레포 클론 릴리스 다운로드

옵션 A - uv 사용 (개발자용)

  1. uv 설치 (Python 3.12+).
  2. 레포를 클론하고 폴더를 엽니다.
  3. uv sync로 의존성을 설치합니다.
  4. uv run python -m app.main 으로 실행합니다.

옵션 B - 릴리스 바이너리 (일반 사용자용)

  1. 최신 빌드를 다운로드합니다.
  2. 압축 해제 후 실행 파일을 시작합니다.
  3. 뮤직 라이브러리 경로를 설정한 뒤 분석과 익스포트를 진행합니다.
